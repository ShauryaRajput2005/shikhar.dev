## 🌟 Overview

Welcome to my personal portfolio repository! This project is a modern web application built to showcase my skills, experience, projects, and educational background. It features smooth animations, a clean user interface, and responsive design, ensuring a great user experience across all devices.

### 🔗 Live Link
👉 **[https://shikhar-dev-smoky.vercel.app/](https://shikhar-dev-smoky.vercel.app/)**

## ✨ Sections Included

- **Hero:** Engaging introduction with a call-to-action.
- **About:** A brief overview of who I am and what I do.
- **Experience:** My professional journey and past roles.
- **Projects:** A curated list of my top projects with descriptions and links.
- **Skills:** A visual representation of my technical stack and tools.
- **Education:** My academic background and qualifications.
- **Certifications:** Relevant certifications I've earned.
- **Achievements:** Notable milestones and accomplishments.

## 🛠️ Getting Started

Follow these instructions to set up the project locally on your machine.

### Prerequisites

- Node.js (v18.17 or higher recommended)
- npm, pnpm, yarn, or bun

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ShauryaRajput2005/shikhar.dev.git
   ```

2. **Navigate to the directory**
   ```bash
   cd portfolio
   ```
   *(Note: Adjust the directory name if cloned directly from GitHub with a different name).*

3. **Install dependencies**
   ```bash
   npm install
   # or yarn install / pnpm install / bun install
   ```

4. **Run the development server**
   ```bash
   npm run dev
   # or yarn dev / pnpm dev / bun dev
   ```

5. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the application running.
